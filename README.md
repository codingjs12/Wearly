### 외부 API
- ⛅  **기상청 API:** 위치 기반 날씨 API
- 🔐 **Google 로그인 API:** Google 계정을 통한 소셜 로그인 기능
- 📩 **이메일 인증 API:** 회원가입 시 이메일 인증을 위한 이메일 인증 연동
- 🟢 **Naver 로그인 SDK:** Naver 계정을 통한 소셜 로그인 기능
- 🟡 **Kakao 로그인 API:** Kakao 계정을 통한 소셜 로그인 기능
